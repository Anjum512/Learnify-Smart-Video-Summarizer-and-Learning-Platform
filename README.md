# Learnify – Smart Video-to-Text Learning Assistant

Learnify is an AI-based learning assistant designed to help students convert educational video and audio content into structured text notes, summaries, quizzes, and learning materials for easier understanding and revision.

---

## Features

- AI-based Video-to-Text Conversion
- Smart Educational Summaries
- Quiz Generation System
- User Authentication
- OTP Verification
- Forgot Password System
- Student Dashboard
- Search Functionality
- Responsive Modern UI
- Interactive Learning Experience
